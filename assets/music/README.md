# Background Music - Psychological Hooks

Add ambient, tension-building music tracks here for psychological engagement.

## Required Music Types:

### 1. Dark Ambient (for shame/regret videos)
- Slow, minor key
- No vocals
- Creates unease
- Examples: "Dark Ambient Piano", "Melancholic Strings"

### 2. Tension/Suspense (for revelation videos)  
- Building tension
- Sparse, mysterious
- Keeps viewers on edge
- Examples: "Mystery Piano", "Suspenseful Background"

### 3. Contemplative (for realization videos)
- Thoughtful, slow
- Hopeful undertones
- Reflective mood
- Examples: "Thoughtful Piano", "Contemplative Strings"

## File Naming (Important):

Name files by emotion for automatic matching:

- `tension_1.mp3`, `tension_2.mp3` - For suspenseful content
- `regret_1.mp3`, `regret_2.mp3` - For shame/mistake content  
- `dark_1.mp3`, `dark_2.mp3` - For serious revelations
- `calm_1.mp3`, `calm_2.mp3` - For neutral content

Bot will automatically pick matching music based on script emotion.
